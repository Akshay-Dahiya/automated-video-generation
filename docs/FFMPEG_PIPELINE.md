# FFmpeg Video Rendering Pipeline

## Overview

The FFmpeg pipeline handles the final composition of short-form vertical videos (9:16 aspect ratio, 1080x1920 resolution). It combines multiple video clips, voiceover audio, background music, and animated subtitles into a single polished output.

## Pipeline Stages

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  1. PREPARE     │───▶│  2. CONCATENATE  │───▶│  3. MIX AUDIO   │
│  Scale & Crop   │    │  Join Clips      │    │  Voice + Music   │
│  to 9:16        │    │  Sequentially    │    │  Volumes         │
└─────────────────┘    └─────────────────┘    └─────────────────┘
                                                        │
                                                        ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  6. UPLOAD      │◀───│  5. OPTIMIZE     │◀───│  4. SUBTITLES   │
│  CDN Storage    │    │  Final Encode    │    │  Burn ASS/SRT   │
│                 │    │  Compression     │    │  Into Video      │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

## Stage 1: Prepare Clips (Scale & Crop to 9:16)

Each source clip is scaled and center-cropped to fit the vertical format.

```bash
ffmpeg -y -i input_clip.mp4 \
  -vf "scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920,setsar=1" \
  -c:v libx264 -preset fast -crf 20 \
  -an -r 30 \
  output_prepared.mp4
```

**Key parameters:**
- `scale=1080:1920:force_original_aspect_ratio=increase` - Scale up maintaining ratio
- `crop=1080:1920` - Center crop to exact dimensions
- `setsar=1` - Set sample aspect ratio to square pixels
- `-an` - Strip audio from source clips
- `-r 30` - Normalize to 30fps

## Stage 2: Concatenate Clips

Join prepared clips sequentially using concat demuxer.

```bash
# Create concat list file
echo "file 'clip_01_prepared.mp4'" > concat_list.txt
echo "file 'clip_02_prepared.mp4'" >> concat_list.txt
echo "file 'clip_03_prepared.mp4'" >> concat_list.txt

# Concatenate
ffmpeg -y -f concat -safe 0 -i concat_list.txt \
  -c copy \
  concatenated.mp4
```

## Stage 3: Mix Audio (Voice + Background Music)

Combine voiceover with background music at specified volumes.

```bash
# Voice + Background Music
ffmpeg -y \
  -i concatenated.mp4 \
  -i voiceover.mp3 \
  -i background_music.mp3 \
  -filter_complex "[1:a]volume=1.0[voice];[2:a]volume=0.15[music];[voice][music]amix=inputs=2:duration=shortest[aout]" \
  -map 0:v -map "[aout]" \
  -c:v copy -c:a aac -b:a 192k \
  -shortest \
  mixed_output.mp4
```

**Audio mixing tips:**
- Voiceover: volume=1.0 (full)
- Background music: volume=0.10-0.20 (subtle)
- Use `amix` with `duration=shortest` to match voiceover length
- `-shortest` ensures video matches audio length

## Stage 4: Burn Subtitles (ASS Format)

Add animated word-by-word subtitles using the ASS format.

```bash
ffmpeg -y \
  -i mixed_output.mp4 \
  -vf "ass=subtitles.ass" \
  -c:v libx264 -preset medium -crf 23 \
  -c:a copy \
  final_with_subs.mp4
```

### ASS Subtitle Format (Word Highlighting)

```ass
[Script Info]
Title: Auto-Generated Subtitles
PlayResX: 1080
PlayResY: 1920

[V4+ Styles]
Style: Default,Montserrat Bold,24,&H00FFFFFF,&H000000FF,&H00000000,&H00000000,-1,0,0,0,100,100,0,0,1,3,2,2,40,40,120,1

[Events]
Dialogue: 0,0:00:00.00,0:00:02.50,Default,,0,0,0,,{\kf50}Did {\kf40}you {\kf45}know {\kf60}that...
Dialogue: 0,0:00:02.50,0:00:05.00,Default,,0,0,0,,{\kf55}AI {\kf50}can {\kf45}now {\kf60}generate {\kf50}videos
```

The `\kf` tag creates karaoke-style fill effect (word highlights as it's spoken).

## Stage 5: Final Optimization

Optimize the output for social media platforms.

```bash
# YouTube Shorts / Instagram Reels optimized
ffmpeg -y \
  -i final_with_subs.mp4 \
  -c:v libx264 -preset medium -crf 23 \
  -profile:v high -level:v 4.1 \
  -pix_fmt yuv420p \
  -c:a aac -b:a 192k -ar 44100 \
  -movflags +faststart \
  -metadata title="Video Title" \
  -metadata comment="Generated by VideoGen AI" \
  output_final.mp4
```

**Platform requirements:**
| Platform | Max Duration | Resolution | Max Size |
|----------|-------------|-----------|----------|
| YouTube Shorts | 60s | 1080x1920 | 256MB |
| Instagram Reels | 90s | 1080x1920 | 4GB |
| TikTok | 180s | 1080x1920 | 287.6MB |

## Advanced Filter Chains

### Ken Burns Effect (Zoom/Pan on Images)

```bash
ffmpeg -y -loop 1 -i image.jpg -t 5 \
  -vf "scale=1200:2134,zoompan=z='min(zoom+0.0015,1.5)':d=150:x='iw/2-(iw/zoom/2)':y='ih/2-(ih/zoom/2)':s=1080x1920" \
  -c:v libx264 -pix_fmt yuv420p \
  ken_burns_clip.mp4
```

### Fade Transitions Between Clips

```bash
ffmpeg -y \
  -i clip1.mp4 -i clip2.mp4 \
  -filter_complex "[0:v]fade=t=out:st=4:d=1[v0];[1:v]fade=t=in:st=0:d=1[v1];[v0][v1]concat=n=2:v=1:a=0" \
  -c:v libx264 -crf 23 \
  faded_output.mp4
```

### Color Grading (Cinematic Look)

```bash
ffmpeg -y -i input.mp4 \
  -vf "eq=contrast=1.1:brightness=0.02:saturation=1.2,curves=preset=cross_process" \
  -c:v libx264 -crf 23 \
  color_graded.mp4
```

### Adding Watermark/Logo

```bash
ffmpeg -y \
  -i video.mp4 -i logo.png \
  -filter_complex "[1:v]scale=80:-1,format=rgba,colorchannelmixer=aa=0.7[logo];[0:v][logo]overlay=W-w-20:20" \
  -c:v libx264 -crf 23 -c:a copy \
  watermarked.mp4
```

## Complete Pipeline Command (Single Pass)

For production efficiency, the entire pipeline can be combined:

```bash
ffmpeg -y \
  -f concat -safe 0 -i concat_list.txt \
  -i voiceover.mp3 \
  -i background_music.mp3 \
  -filter_complex \
    "[0:v]scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920,setsar=1,fps=30[scaled];\
     [1:a]volume=1.0[voice];\
     [2:a]volume=0.15,afade=t=in:d=2,afade=t=out:st=55:d=5[music];\
     [voice][music]amix=inputs=2:duration=first[audio];\
     [scaled]ass=subtitles.ass[final]" \
  -map "[final]" -map "[audio]" \
  -c:v libx264 -preset medium -crf 23 \
  -profile:v high -pix_fmt yuv420p \
  -c:a aac -b:a 192k \
  -movflags +faststart \
  -shortest \
  output_final.mp4
```

## GPU Acceleration (Optional)

For NVIDIA GPUs with NVENC:

```bash
ffmpeg -y -hwaccel cuda -hwaccel_output_format cuda \
  -i input.mp4 \
  -vf "scale_cuda=1080:1920" \
  -c:v h264_nvenc -preset p4 -cq 23 \
  output.mp4
```

This reduces rendering time by 3-5x on supported hardware.
